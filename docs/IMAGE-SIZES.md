# Image Sizes Reference

Use these dimensions when preparing or replacing images for the ROLCC website. Placeholder images or colored divs can be used until final assets are supplied.

| Component                    | Size        | Aspect | Usage                          |
| ---------------------------- | ----------- | ------ | ------------------------------ |
| Marquee hero strip           | 400×530 px  | 3:4    | Each `marquee-hero__img`       |
| ghc feature cards            | 500×400 px  | 5:4    | `ghc__card-img`                |
| Impact carousel              | 1200×500 px | 12:5   | `impact-carousel__slide`       |
| ministry-nav-card            | 600×400 px  | 3:2    | Explore Other Ministries cards |
| River Kids gallery (if kept)  | 400×300 px  | 4:3    | Gallery grid                   |
